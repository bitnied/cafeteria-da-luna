# ☕ Cafeteria da Luna

Cardápio digital + controle de estoque para brincar de cafeteria em casa.
Otimizado para tablet (iPad) e celular (iPhone).

## Como usar
- **Cardápio:** escolha as bebidas e a quantidade (− / +), toque em **🛎️ Pedir**, confira o resumo e toque em **Entregar pedido**. O estoque baixa sozinho.
- **Estoque:** ajuste as quantidades, adicione novas cápsulas (✏️) ou outras bebidas.
- **Backup:** na aba Estoque, use **Salvar backup** de vez em quando (gera um arquivo) e **Restaurar backup** se trocar de aparelho.

## Onde os dados ficam
Tudo é salvo **no próprio aparelho/navegador** (não precisa de internet depois de instalado). Por isso o backup é importante: cada aparelho tem seus próprios dados.

## Instalar na tela inicial (iPad/iPhone)
Abra o site no **Safari** → botão Compartilhar → **Adicionar à Tela de Início**.
Vira um ícone de app e abre em tela cheia.

## Tecnologia
HTML + CSS + JavaScript puro, PWA com service worker (funciona offline). Hospedado no GitHub Pages.
